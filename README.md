# 🏜️ 废土商队模拟器

基于 React + Node.js + Express 的废土商队经营模拟游戏。

## ✨ 游戏特色

- 🚚 **创建商队**：自定义商队名称、领袖和起始城市
- 🏪 **采购货物**：9种货物（净水、干粮、药品、燃料、武器、钢材、废料、黑市药品、布料）
- 🗺️ **跨城运输**：5座废土城市，7条商路，每条路线有不同距离和危险度
- ⚡ **随机遭遇**：10种随机事件（强盗袭击、沙暴、发现宝藏、友好旅人等）
- 📊 **动态价格**：价格随城市库存、供需关系波动
- 🎒 **负重体力**：商队有最大负重和体力限制，需要休整恢复
- 📈 **资金曲线**：实时查看资金变化趋势
- 📜 **交易日志**：完整记录所有交易和事件

## 🏙️ 城市介绍

| 城市 | 特色 |
|------|------|
| 灰烬堡垒 | 军火与钢铁产地，武器钢材便宜 |
| 绿洲镇 | 农业发达，净水干粮价格低 |
| 铁锈港 | 废料堆积如山，回收物资便宜 |
| 辐射城 | 黑市药品产地，风险高利润大 |
| 沙丘集市 | 游牧民聚集地，布料药材多 |

## 🚀 启动方式

### 前置要求
- Node.js >= 14
- npm

### 安装依赖
```bash
# 安装后端依赖
npm install

# 安装前端依赖
cd client
npm install
cd ..

# 或者一键安装
npm run install-all
```

### 开发模式运行（前后端同时启动）
```bash
npm run dev
```

### 生产模式运行
```bash
# 先构建前端
npm run build

# 启动服务器
npm start
```

启动后访问：http://localhost:5000

## 🎮 游戏玩法

1. **创建商队**：填写商队名称、领袖姓名，选择起始城市
2. **观察市场**：查看各城市价格差异，寻找套利机会
3. **采购货物**：在低价城市买入货物（注意负重限制）
4. **跨城运输**：选择目的地城市出发（消耗体力、水、食物、燃料）
5. **应对事件**：旅途中可能遭遇各种随机事件，做出选择
6. **出售结算**：到达高价城市后卖出货物赚取差价
7. **休整恢复**：体力不足时在城市休整恢复

## 🎯 游戏策略提示

- 💡 绿洲镇买入水和食物 → 其他城市卖出
- 💡 灰烬堡垒买入武器钢材 → 其他城市卖出
- 💡 铁锈港买入废料 → 其他城市卖出
- 💡 辐射城买入黑市药品 → 沙丘集市卖出（注意高风险）
- 💡 随身携带足够的水、食物、燃料和武器
- 💡 高危险度路线更可能遇到强盗，但距离可能更近
